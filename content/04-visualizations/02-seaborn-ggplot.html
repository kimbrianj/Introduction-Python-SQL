Other Plotting Libraries#

While matplotlib can be very useful for creating graphs, it can be easy to get bogged down in all of the intricacies of customizing everything you want to do. In this section, we introduce a

Seaborn#

Seaborn is a package built on top of matplotlib that is meant to make some of the difficult tasks easier to do. We first import it then set the style. We are using the default style for Seaborn here.

import seaborn as sns

sns.set_style()

A basic histogram can be made using the sns.displot function, passing the Data Frame as the first argument, then specifying the x variable and any other arguments to adjust the plot as necessary.

sns.displot(df_2015, x = 'CA01', stat="density", discrete=True)

<seaborn.axisgrid.FacetGrid at 0x1328ad910>

This doesn’t look very nice because it’s trying to plot every single value, and there are some big outliers. Let’s limit to a small area.

sns.displot(df_2015, x = 'CA01', stat="density", discrete=True).set(xlim=(0,50))

<seaborn.axisgrid.FacetGrid at 0x1399eb100>

ggplot#

For people familiar with R and ggplot2, using ggplot might be preferable because it uses a lot of the same syntax as the R version. The ggplot visualizations use the grammar of graphics to build a plot, meaning it requires defining the dataset, the aesthetics of the plot, as well as the geometric objects to use to build the plot. For example, to create a histogram, we would need to specify the Data Frame we are using with ggplot, add on the aesthetics using aes, then use the appropriate geom_histogram to add the histogram geometric object on top of that.

from plotnine import *

ggplot(df_2015) + aes(x = 'CA01') + geom_histogram(bins = 20)
